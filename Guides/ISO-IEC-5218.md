# ISO/IEC 5218

Code for representation of human sex (male or female)

ISO/IEC 5218 Information technology â€” Codes for the representation of human sexes is an international standard that defines a representation of human sexes through a language-neutral single-digit code. It can be used in information systems such as database applications.

The four codes specified in ISO/IEC 5218 are:

- 0 = Not known;
- 1 = Male;
- 2 = Female;
- 9 = Not applicable.
